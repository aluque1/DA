# Conjuntos mediante Arboles AVL

``` c++

 bool insert(T const& e) // Insertar elemento : O(log N)
 bool erase(T const& e)  // Eliminar elemento : O(log N)
 bool contains(T const& e) const // Averiguar la pertenencia al cjto : O(log N)
 bool empty() const // Averiguar si el cjto es vacio : O(1)
 int size() const// Obtener el cardinal de cjto : O(1)
```
